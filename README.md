# 🛰️ Strategic Intelligence Hub (SIH)

An enterprise-grade Business Intelligence platform for SMEs that combines **Deterministic Python Analytics** with **AI Strategic Reasoning**. Built using the "Prompt Firewall" architecture to ensure hallucination-free reporting.

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.9+
- A Google Gemini API Key ([Get one here](https://aistudio.google.com/))

### 2. Installation
```bash
# Clone the repository
git clone <your-repo-url>
cd sis

# Install dependencies
pip install streamlit pandas plotly scikit-learn google-generativeai fpdf2
```

### 3. Run Locally
```bash
streamlit run app.py
```

## System Modules
- **Chapter 1: Performance Pulse:** Real-time KPI tracking and anomaly detection.
- **Chapter 2: Visual Intelligence:** Trend analysis, correlation heatmaps, and distribution audits.
- **Chapter 3: ML Behavior Drivers:** Random Forest and Linear Regression feature importance.
- **Chapter 4: AI Insights Engine:** Detailed narrative analysis and forecasting explainers.
- **Chapter 5: Strategic Advice:** Actionable recommendations and risk assessments.
- **Chapter 6: Forecasting & Model Audit:** Integrated predictive line plots and statistical metrics (R2, RMSE, MAE).
- **Chapter 7: Comprehensive Conclusion:** Executive wrap-up generated by AI.

## Architecture: The Prompt Firewall
SIH uses a strict **Separation of Concerns**:
1. **Deterministic Layer (Pandas/Scikit-Learn):** Performs all math, statistical audits, and forecasting.
2. **AI Layer (Gemini 1.5 Flash):** Acts only as a strategic interpreter. It is forbidden from performing math to prevent numerical hallucinations.

## Data Management
- **JSON Exports:** Versioned analysis results for integration.
- **PDF Strategy Reports:** High-quality executive briefs with integrated charts.
- **Diagnostics:** Real-time FinOps (cost monitoring) and API health checks.
